2. Description du service

AutoReparis OS est une application professionnelle de gestion d'atelier automobile permettant :

  • La gestion et le suivi des interventions de réparation
  • La gestion du stock de pièces détachées avec scanner de codes-barres
  • Le planning et la gestion de l'équipe de mécaniciens
  • La gestion de la base clients et de leurs véhicules
  • Le suivi en temps réel des réparations pour les clients
  • La réception des demandes de devis depuis le site web
  • Les notifications push pour le suivi d'activité
